💼 Top-Paying Data Analyst Jobs & Skills in 2024 | SQL-Powered Insights
📊 A curated SQL analytics project exploring the highest-paying remote Data Analyst roles, their required skills, and the tech that commands top salaries—extracted from real-world job posting data.

🚀 Project Overview
In an increasingly competitive job market, knowing where the top salaries are—and what skills they demand—can be a game-changer. This project uses advanced SQL on a job postings dataset to surface:

💰 Top 10 highest-paying remote Data Analyst roles

🛠️ Most in-demand and highest-paying skills

📌 Company-level trends in salary, skillset, and tech stack

All powered by clean, efficient SQL scripts—focused on clarity and insight.

📁 Repository Structure
File	Description
1_top_paying_jobs.sql	Retrieves the top 10 highest-paying remote Data Analyst jobs
2_skills_required_for_top_jobs.sql	Reveals the specific skillsets required for these top-paying roles
3_top_demanded_skills.sql	Identifies the 10 most frequently required skills for Data Analyst roles
4_top_paying_skills.sql	Calculates the average salary associated with each skill
5_most_optimal_skills.sql	Identifies skills that balance high demand and high pay—best of both
